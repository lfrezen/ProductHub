# ProductHub

O ProductHub é um sistema simples de gerenciamento de produtos, desenvolvido com o objetivo de demonstrar habilidades técnicas em backend (.NET) e frontend (Angular).

## Visão Geral

O sistema contempla:

- Autenticação e autorização de usuários
- Gerenciamento de produtos (CRUD)
- Processo em background para atualização do status de estoque
- API REST desenvolvida em .NET
- Aplicação web desenvolvida em Angular

## Tecnologias Utilizadas

### Backend

- .NET
- Entity Framework Core
- Autenticação via JWT

### Frontend

- Angular
- TypeScript

## Arquitetura

O backend foi estruturado seguindo uma arquitetura em camadas, com separação de responsabilidades entre:

- API
- Application
- Domain
- Infrastructure

Essa abordagem facilita a manutenção, a testabilidade e a evolução do sistema.
